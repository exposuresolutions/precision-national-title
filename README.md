# Precision National Title Website

This repository contains the source code for the Precision National Title website.

## Files
- `src/index.html`: The main landing page (formerly `precision-nation-upgraded.html`)
- `src/services.html`: The services page (formerly `precision-services-upgraded.html`)
- `src/about.html`: The about page (formerly `precision-about-upgraded.html`)

## Usage in GoHighLevel (GHL)

To use these files in GHL:
1. Copy the raw content of the HTML files.
2. Paste it into a "Custom Code" element in your GHL funnel/website builder.

Alternatively, you can host these files via GitHub Pages or direct raw links if your setup supports it, but copying the code ensures maximum compatibility with GHL's builder adjustments.
